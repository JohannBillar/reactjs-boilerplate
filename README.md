# ReactJS Project Boilerplate
## Repo created to quickly setup my React projects.
 
### Quick start a project with:
* Webpack and webpack-dev-server
* Babel 6
* ESLint (extended with airbnb)
* Jest
* SASS
* Express
* React-Router
* Much more coming...

___
### Usage:
```sh
$ npm install
$ npm test
$ npm start or $ NODE_ENV=production npm start
```
